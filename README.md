# embeddenator-testkit

Test utilities and fixtures for `embeddenator`.

This crate is a small helper library used by integration/contract tests and benchmark harnesses.

## Status

Alpha. APIs may change.

## Usage

Use as a dev dependency inside workspace crates:

```toml
[dev-dependencies]
embeddenator-testkit = { path = "../embeddenator-testkit" }
```
